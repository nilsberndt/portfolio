ConCSStruct (concsstruct.com)


ConCSStruct is a coding-based game where users complete construction tasks by coding "blueprints" using HTML and modular CSS class styles. The game is built with a React / Redux state storage system providing real-time updating, game logic and level management.

10kT (10kt.app)


10kT is a dynamic, real-time text editing platform where {wordblock}s can be modified by users from the Android app and viewed live on the web at 10kt.app. Changes are seen immediately across all channels due to a Google Firebase Realtime Database back-end.

Now in Beta testing on the Google Play Store.
